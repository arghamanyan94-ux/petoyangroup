# Artik Auto Service — Website Single-file trilingual (Armenian / Russian / English) website for an oil-change and autogas (LPG) installation garage in Artik, Armenia. ## Publish it on GitHub Pages 1. Create a new GitHub repository (e.g. `artik-auto-service`). 2. Upload `index.html` to the root of the repository (drag & drop on github.com works fine, or via git): ``` git init git add index.html git commit -m "Add site" git branch -M main git remote add origin https://github.com//.git git push -u origin main ``` 3. In the repo, go to **Settings → Pages**. 4. Under **Source**, choose the `main` branch and `/ (root)` folder, then **Save**. 5. GitHub will publish the site at: `https://.github.io//` (usually live within a minute or two). ## What's inside - **Language switcher** (top right): Armenian is the default/main language, with Russian and English toggles. The choice is remembered on return visits. - **Header**: logo on the left, navigation, click‑to‑call phone number. - **Hero**: full-bleed banner image with call-to-action buttons. - **Services**: two "bay" cards — Oil Change and Autogas Installation — each with its own photos and details. - **Why Autogas**: benefit highlights (savings, warranty, eco-friendliness, professional installation). - **Gallery**: mosaic of garage photos. - **Contact**: address (Artik, Shahumyan St., 3rd Lane), phone (+374 94 353541), hours, and an embedded map. ## Customizing - All text lives in the `translations` object near the bottom of `index.html` (three blocks: `hy`, `ru`, `en`) — edit copy there. - Colors and fonts are set as CSS variables at the top of the `
